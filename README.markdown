LG Lucid VS840 系统修改器
=========================
基于xposed框架,对lg vs840原生rom进行简单的修改.

修改列表
-------
* 添加*系统选择*到*设置*
* 从*设置*里删除*Backup Assistant Plus*
* 从*帐号与同步*里删除*VERIZON帐户选项*
* 删除状态栏的三角形漫游图标
* 删除定位的十字架图标

安装方法
--------
1. root手机
2. 安装xposed
3. 安装XposedLucidMod
